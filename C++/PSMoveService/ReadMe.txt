En:
Requires PSMoveService and configuration.
For a 32-bit operating system, copy the contents of the "..\TrueOpenVR\Additionally\PSMoveService\x86" folder to the "C:\Windows" folder.
For a 64-bit operating system, copy the contents of the "..\TrueOpenVR\Additionally\PSMoveService\x64" folder to the "C:\Windows" folder.
